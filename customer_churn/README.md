# Customer Churn Prediction System

## Project Overview
This project predicts whether a customer will leave a company (churn) based on their behavior and usage data. It uses machine learning techniques to analyze customer data and provide real-time churn predictions.  

Key highlights:
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering (tenure, usage patterns, etc.)  
- Machine Learning models: Random Forest and Logistic Regression  
- Model evaluation: accuracy, precision, recall  
- Interactive dashboard using Streamlit
