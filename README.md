2) Customer Churn Prediction System:
A. Project Overview
This project predicts whether a customer will leave a company (churn) based on their behavior and usage data. It uses machine learning techniques to analyze customer data and provide real-time churn predictions.

B. Features:

a) Data cleaning and preprocessing
b) Exploratory Data Analysis (EDA)
c) Feature engineering (tenure, usage patterns, etc.)
d) Machine Learning models: Random Forest and Logistic Regression
e) Model evaluation: accuracy, precision, recall
f) Interactive dashboard using Streamlit
